BADGE STAGE SYSTEM

같은 단계라도
안쪽 아이콘이 진화합니다

출석, 거리, 시간, 효율, 성장, 시즌, 갤 전체, 보너스는 카테고리 상징은 유지하되 해금 단계가 올라갈수록 안쪽 아이콘 자체가 바뀌는 배지 시스템 시안입니다. 같은 카테고리 안에서도 Lv.1, Lv.2, Lv.3가 서로 다른 그림으로 보이게 하고, 단계가 오를수록 안쪽 심볼의 형태와 장식이 같이 진화하도록 잡았습니다.

같은 tier = 같은 단계 언어 카테고리별 차이 = 내부 상징 높은 tier = 안쪽 아이콘 진화 1~8 코어 단계 + 9~12 시즌 확장 모바일 축소 시에도 단계 인지 가능
입문 구간 도약 구간 대표 구간 하이라이트 구간

핵심 규칙

단계는 아이콘 진화로, 파트는 상징으로

현재 데이터 기준으로 배지 이름과 설명은 그대로 유지하고, UI에서 보이는 안쪽 심볼의 조형을 단계 공통 규칙으로 다시 설계하는 방향입니다.

tier 1 첫 해금, 입문, “시작했다”는 느낌
tier 4 중반 도약, 구조가 복합화되는 단계
tier 8 최상위, 대표 칭호에 가까운 인상
tier 12 시즌 확장, 연간 전설급 마감용
