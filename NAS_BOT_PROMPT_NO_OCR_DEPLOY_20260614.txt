Deployment steps:

1. SSH into NAS.
2. Go to deployment directory.

```sh
cd /volume1/docker/swimdash
```

3. Fetch latest source and checkout branch.

```sh
git fetch origin
git checkout feature/cartoon-dot-swim-dashboard-domain-ocr
git pull --ff-only origin feature/cartoon-dot-swim-dashboard-domain-ocr
```

4. Rebuild and restart container.

```sh
docker compose up -d --build
```

5. Confirm container health.

```sh
docker compose ps
docker logs swimdash-app --tail 200
curl -I http://127.0.0.1:8766/
```

6. Rebuild public JSON without OCR.

```sh
docker exec swimdash-app python -m swimdash rebuild
```

7. Run safe refresh without OCR.

```sh
docker exec swimdash-app python -m swimdash refresh --skip-ocr
```

8. Verify primary domain pages with cache busting.

Rollback plan:

If https://swim.h19h19.com fails but the container works:
- Do not stop the container.
- Disable or revert only the Synology reverse proxy rule for swim.h19h19.com.
- Use http://swimmingdiary.duckdns.org:8766/ as rollback.

If the container fails:

```sh

docker logs swimdash-app --tail 200
```
